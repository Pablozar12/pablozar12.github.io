Introducción a HTML

HTML Boilerplate

Es la estructura básica de toda la página web. En el programa Visual Studio Code, escribimos html y hacemos clic en html:5.

Encabezados o títulos por nivel, de h1 a h6

Empleamos esto para destacar el texto por importancia.

Esto está hecho con h3

Salto de línea con br

Esto se usa para crear un salto de línea cuando no quieres que aparezca todo en el mismo renglón hasta el límite de la página.
Un ejemplo claro son
los poemas que hay en páginas web.

Énfasis de texto: Cursiva

Este sirve para poner texto en cursiva.
Ejemplo: coche rojo.
Esto ha sido posible poniendo <em>, el texto o la palabra que queramos poner en cursiva
y cerramos cursiva con </em>.

Énfasis de texto: Negrita

Este es parecido a poner texto en cursiva pero en este caso lo ponemos en negrita, que va a resaltar más.
Ejemplo: la moto es cara.
El procedimiento es el mismo pero en vez de usar "em" usamos "strong".

Lista desordenada

Se usa cuando quieres hacer una lista para colocar, pero no buscas un orden, es decir, no necesitas que algo vaya antes que lo otro.
Para usarlo se escribe <ul>, escribes lo que quieres colocar con un comando(<li>) que se mencionará mas adelante, y se cierra con </ul>.

Lista ordenada

Se emplea igual que en la lista desordenada, pero este coloca por orden (1. 2. 3. etc) las ideas o cosas que escribas.
Para usarlo tienes que hacer lo mismo que en el anterior pero en vez de escribir "ul" escribes "ol".

Viñietas

Se usa para poner ideas o cosas de manera desordenada(por puntos) u ordenadas(numerado).
Ejemplo de listas con viñietas:

Adornos para navidad

  • Estrella Fugaz
  • Árbol de Navidad
  • Belén montable

Mejores marcas de móvil

  1. Samsung
  2. Apple
  3. Huawei

Inserción de imágenes, texto si imagen no carga, resolución(px)

Lo que hacemos con esto es insertar imágenes sacadas de internet cogiendo la URL.
