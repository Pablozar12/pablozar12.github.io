Comentarios

Esto sirve para escribir algo y que el navegador lo ignore y
sólo se pueda ver buscando el código fuente de la página.
El comando se escribe así:
<!-- El texto que quieres que el navegador ignore! -->.
Para acceder al código funte de la página solo tienes que darle a F12
ó pulsando ctrl + mayús + i.
